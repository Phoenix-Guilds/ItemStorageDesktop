# ItemStorage Manager (Phoenix Nest Edition)

Утилита для автоматической синхронизации базы данных аддона **ItemStorageBrowser** между игроками гильдии.

## Основные функции
* **Режим Админа:** Автоматически сканирует логи аддона `CharacterStatusLogger` (используется помощниками по складу). При обнаружении выхода из игры складского персонажа, утилита сверяет локальную базу данных с облаком и выгружает её в GitHub, если локальная версия новее.
* **Режим Пользователя:** Периодически (раз в 15 минут) проверяет наличие обновлений базы данных в облаке. Если в GitHub появилась более свежая версия, утилита автоматически скачивает её и обновляет файл в папке с игрой.
* **Авто-обновление:** При каждом запуске программа проверяет наличие новой версии самого менеджера и предлагает перейти на страницу релиза для скачивания, если новая версия будет найдена.

## Системные требования
* Windows 10/11.
* Установленный аддон **ItemStorageBrowser** (для всех).
* Установленный аддон **CharacterStatusLogger** (только для помощников по складу/админов).

## Использование
1. **Первый запуск:** Укажите полный путь к папке с игрой (например, `C:\Games\World of Warcraft Sirus`).
2. **Для админов:** Введите GitHub Personal Access Token (его можно получить у ГМа гильдии).
3. **Работа:** Программа с версии 0.1.2 работает в псевдо-консольном режиме с цветовой индикацией: зеленый — успех, голубой — информация, красный — ошибки. Окно можно свернуть в трей. Также изменена механика установки новых версий: перед установкой версии 0.1.2, все предыдущие установленные версии нужно удалить вручную, а начиная с версии 0.1.2 программа будет обновлять сама себя при установке новой версии - после этого, ручное удаление старых версий перед установкой новых, не требуется.
4. **Синхронизация:** При первом запуске программа всегда отдает приоритет облачной версии базы данных, чтобы гарантировать актуальность данных.

## Инструкция по установке
1. Перейдите в раздел [Releases](https://github.com/Phoenix-Guilds/ItemStorageDesktop/releases).
2. Скачайте последний доступный файл:
   * **ItemStorageBrowser_..._x64_ru-RU.msi** — стандартный установщик Windows (теперь - только на русском языке).
3. Запустите файл и следуйте инструкциям.
4. **Важно:** Если Windows выдаст предупреждение "SmartScreen", нажмите **"Подробнее"** -> **"Выполнить в любом случае"** (это стандартное поведение для программ без платной цифровой подписи).
